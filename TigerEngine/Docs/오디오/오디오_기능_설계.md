# 오디오 기능 설계

## 채널 그룹 설계
- AudioSourceComponent에 ChannelGroup 속성 추가
- 매니페스트 Group(BGM/SFX/AMB/UI) 자동 적용
- AudioSystem에서 그룹 생성/볼륨 제어 API 제공
- 그룹 이름은 문자열이며 대소문자 구분

## 2D/3D 재생 규칙
- 2D: AudioSourceComponent만 있으면 재생 가능 (리스너 불필요)
- 3D: AudioSourceComponent + 리스너 필요
- 리스너 미부착 시 FMOD 기본 리스너가 원점(0,0,0)에 고정됨
- 3D 공간화 안정성을 위해 모노 파일 권장

## 도플러/거리 스케일 기준
- 도플러는 상대 속도에 민감하므로 단위 스케일을 맞춰야 함
- 엔진 단위가 cm일 경우, velocity를 0.01 배로 변환해 m/s 기준으로 맞춤
- distanceFactor = 0.01 (cm -> m)
- dopplerScale = 1.0 (필요 시 on, 테스트 시 off 가능)
- rolloffScale = 1.0
- 적용 위치: AudioManager::Initialize 이후 (EngineApp::OnInitialize 등)

## 테스트 기준 (체크 포인트)
- 리스너 부착/미부착 비교 (원점 기준 동작 확인)
- 거리 감쇠: 가까움/중간/멀어짐 단계별 볼륨 변화 확인
- 도플러: 접근/이탈 시 피치 변화 확인

## PR 변경 파일 정리 기준
- AudioSystem 채널 그룹/볼륨 API 추가 파일
- AudioSourceComponent ChannelGroup 속성 추가 파일
- 테스트 스크립트 추가/등록 파일
- 테스트 씬 수정 파일
