# 폴더 구조

---

### 최상위 폴더

Base
- Engine에 사용할 기본 데이터 모음

Engine
- Base 내용을 이용해 만든 게임에 사용할 코드 모음 

External
- 외부 프로젝트 모음

WorkSpace
- 클라이언트가 게임에 사용할 코드 모음

### Base

Datas
- 기본 데이터 단위

Entity
- 게임 엔진에 사용할 객체의 기본 단위

Renderer
- 게임 엔진에 사용할 렌더러 api 기본 단위

System
- 엔진 루프에 관여 안하는 시스템 코드 모음

### Engine

App
- 게임 프로그램 루프와 동일한 구성을 사용하는 코드들 ( ex> Imgui 사용하는 Editor )

Components
- 게임을 구성하기 위해 사용하는 컴포넌트 모음 ( 사용자 컴포넌트 제외 )

EngineSystem
- 엔진이 사용할 시스템 모음 ( 객체 등록 관리 등 )

Manager
- 엔진이 사용할 매니저 모음 ( 데이터 접근 등 ) 

Object
- 게임이 사용할 기초 단위 모음 ( GameObject, Component )

RendererPlatform
- `Base/Renderer` 에 있는 IRenderer를 상속받은 그래픽 api를 사용하는 클래스 모음 ( DirectX, Opengl 등 )

RenderPass 
- 게임 렌더링 시 `EngineApp`에 정의된 순서대로 렌더링될 스테이지 세팅 코드가 있는 클래스 모음

RenderQueue
- RenderPass가 처리할 데이터를 모아두는 클래스

Scene
- `GameObject` 관리하는 Scene

Shaders
- RenderPass가 사용하는 셰이더 코드 모음 (.hlsl)

Util
- 편의를 위한 코드 모음
